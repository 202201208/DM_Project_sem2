What is digital image ?

Digital image is collection of pixels ,where each pixel has finite and discrete quantity known as intensity .

Pixel is basic element of digital images.

How to represent digital image?

Digital image can be represented as a matrix . where each element of a matrix is a pixel .

Here x and y are spatial coordinates,and f(x,y) is value of intensity at point (x,y).

Pixel density :

Pixel density tells about how much pixel is there in particular length.

Generally , Pixel per density is measured in pixel per inch (PPI) .

More pixel density impies more resolution.

Bits per pixel (bpp) :

Bits per pixel is a number of bits aquired by a pixel of an image .

Generally greyscale image has 8 bpp and colour image has 24 bpp.

Histogram of image:

Histogram is a graph of intensity to number of pixel having that intensity.

which gives information about how many pixels of image have particular intensity.

In colour ( RGB ) image there are three channel so in color image histogram for different channel there will be a curve .
