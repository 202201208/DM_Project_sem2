Image Transformation

Transformation means changes in size, shape, position and orientation etc. . .

In this transformation, The changes occurs only in Geometry of an image but it’s pixel values remain unchanged.

We can apply different type of operations for Geometric Transformation, But That will only change position of pixel without affecting the pixel intensity values.

We can denote Pixel’s position in cartesian co-ordinate system by (x,y) which is the co-ordinate of pixel of actual input image and we can transform it in (x’,y’) which is the co-ordinate of corresponding pixel of output image by applying different functions on input image

So, basically it will map (x,y) of f(x,y) in g(x’,y’) like this:

x’ = Ax(x,y)

y’ = Ay(x,y)

where Ax(x,y) and Ay(x,y) are both functions, which map (x,y) to (x’,y’).

The basic Geometrical Transformations are:

  1. Translation
  2. Scaling
  3. Rotation
  4. Reflection
  5. Shearing

Translation :

Translation means shifting the image by horizontally or/and vertically by the given value in both direction.

x’=x+∆x ; where ∆x is horizontal translation

y’=y+∆y ; where ∆y is vertical translation

Scaling :

Scaling means resizing an image. It alters the size of an image. It means that we can make image bigger or smaller in x or y direction.

We can resize an image or scale it by given scaling factor.

x’ = Sx*x ; Sx = Scaling factor in x-direction

y’ = Sy*y ; Sy = Scaling factor in y-direction

ROTATION :

Rotating means rotate an image at given angle θ by specified or given axis or point. It means that each pixel of image f(x,y) will be rotated at angle θ.

x’ = xcos θ – ysin θ

y’ = xsin θ + ycos θ

where θ is the angle by which we want to rotate an image

Shearing :

Shearing means changing the size and shape of the 2-D object along x-axis and y-axis

Shear Transformation Slants the shape of an object.

There are two Types of shearing Transformation:

1. X-shearing:

In x-shearing, y co-ordinates will be preserved while x co-ordinates will change it’s position(value).

So vertical line will be tilted towards Right or Left.

x’ = x + Shx*y ; Shx = shearing factor in x-direction
