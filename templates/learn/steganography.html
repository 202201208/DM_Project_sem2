Steganography

steganography means hiding some information in some other message or any other object. There are many types of steganography.

Text steganography: hiding information in text. it is rarely used cause text has very limited memory so limited information can be hide.

Image steganography: hiding information in images. it is widely used because in comparison to text more information can be hide.

Video stenography: hiding information in video. though much more information can be hide in comparison to text and image but not so much popular because it becomes very hard to retrieve information.

Image Steganography

There are many methods for hiding information in images. one of the methods is LSB method. Here we will see how to hide text information using LSB method.

For Grayscale Images :

Though there are 256 different intensity level, but human eye can’t differentiate all intensity level. So generally human eye can hardly differentiate two intensities which are too close to each other.

Generally, Grey scale images have 8 bits per pixel.

if we take two least significant bits and use it to store our information then there can be maximum variation of 22 -1 = 3 intensity level which is hard to differentiate with naked eyes. so, in every 4 pixel there is total 8 bits means 1 byte memory for hiding information. so, if there is an image of dimension 1000 x 1000 then total pixel will be 1000000 so total memory to hide =1000000/4=250000 means we can 250000 characters in the image.

To retrieve information back we will take two LSB from each pixel and then we will get text from that.

example :

if image dimension is 3x3 ,then maximum message length can be 9/4=2.25 = 2 . lets suppose massage is “BD” then binary of message will be “ 0100001001000100 ”

Note: We can use 3 LSB too but then there can be maximum variation of 23 -1 = 7 intensity level which is noticeable.

For Color Images :

In color image there are three color channel R, G and B. So, there are two methods:

Signal Channel LSB :

In this method we use only one-color channel to hide information.

For example, we will use only red color channel. red color channel has 8 bits per pixel. now apply same method as grey scale image, so there will be 2 bits in every pixel for hiding information.

example :

if image dimension is 3x3 ,then maximum message length can be 9/4=2.25 = 2 .lets suppose massage is “BD” then binary of message will be “ 01100010 01100100” And we select red channel for hiding information.

Three Channel LSB :

In this method we use all three channels to hide the information. so, there will be 2 bits from R channel, 2 bits from G channel and 2 bits from B channel. so, there will be total 6 bits per pixel to hide the information. So, we can hide 3 times information than the first one. but here visual change can be noticeable sometimes.

example :

if image dimension is 3x3 ,then maximum message length can be ( 9 * 6 ) / 8 = 6.75 = 6.

Let suppose messenger is “HELP” then binary of messge will be “011010000110010101101100 01110000”

From above example, we can see that is almost impossible to differentiate images with hidden data and without hidden data.
