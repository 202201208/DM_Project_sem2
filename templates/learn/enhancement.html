Image Enhancement

Image enhancement is a process of improving appearance of digital image and making image suitable for further analysis . For example improving contrast and brightness ,removing noise ,sharpening images etc..

Image enhancement can be done in two domain :

  1. Spatial Domain
  2. Frequency Domain

SPATIAL DOMAIN

In this domain we directly work with original images .We do some operations on pixels directly.

Spatial domain process is defined by

g(x,y) = T[f(x,y)]

here, f is input image and g is output image.

operator T is defined over a neighborhood of point (x,y) of image f. neighborhood size can be 1x1 , 3x3 ,5x5 etc. . .

INTENSITY TRANSFORMATION

When neighborhood size is 1x1 then operation is called intensity transformation or point processing as output of function is dependent only on that pixel.

here are some example of intensity transformation

  1. image negative
  2. log transformation
  3. power law transformation /gamma transformation
  4. piece wise linear transformation

Image Negative :

Operation for this is : T (r) = (L - 1) - r

where L = 2bits per pixel

Example :

Let suppose we have an RGB color image which has 24 bits per pixel value . 8 bit per pixel for each channel then we will subtract all channel intensity from 255 to get negative .

It is obvious that S will have three component SR , SG , SB same with R.

Where T operator performs following task :

SR = (LR-1 ) - RR

SG = (LG-1 ) - RG

SB = (LB-1 ) - RB

Sx = intensity of channel x in image S.

Lx = bits per pixel value for channel x.

Use Image Negative Tool
LOG TRANSFORMATION :

For an image log transformation is defined as below :

s = c*log( 1+ r)

where c is constant , generally we take

c = (L - 1)/log(1 + MAX - V AL)

because for this value of c output lies between 0 to L - 1 ,means all intensities covered by output, hence range of intensity is covered by output hence range of intensity is covered optimally

In graph you can see that narrow range of lower intensity is extended to wider range and wider range of higher intensity is compressed into narrow range.So images which have majorly lower intensity should be given as input

POWER LAW TRANSFORMATION / GAMMA TRANSFORMATION :

power law transformation or gamma transformation can be define as below

s = crr

Where c and τ are constant.

Generally c =(L-1) , because we want to use range of intensity optimally.

From the above graph you can see that for τ < 1 function behaves similarly as log transformation and for τ> 1 behaves exactly opposite .

FREQUENCY DOMAIN

In spatial domain we deal with images directly .but in frequency domain we deal with rate at which the pixel values are changing in spatial domain.

Here,we first convert image in frequency domain then we perform operation on it and convert result into spatial domain .

There are many method to convert image in frequency domain like Fourier series ,Fourier transformation ,Laplace transformation , Z transformation but here we will see about Fourier transformation only.

FOURIER TRANSFORMATION :

According to Fourier transformation any non periodic signals whose area under the curve is finite can be represented as integrals of the sines and cosines .after being multiplied by a certain weight
