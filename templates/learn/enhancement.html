Image Enhancement

Image enhancement is a process of improving appearance of digital image and making image suitable for further analysis . For example improving contrast and brightness ,removing noise ,sharpening images etc..

Image enhancement can be done in two domain :

  1. Spatial Domain
  2. Frequency Domain

Spatial Domain

In this domain we directly work with original images .We do some operations on pixels directly.

Spatial domain process is defined by

g ( x, y ) = T [ f ( x, y) ]

here, f is input image and g is output image.

operator T is defined over a neighborhood of point (x,y) of image f. neighborhood size can be 1x1 , 3x3 ,5x5 etc. . .

Intensity Transformation

When neighborhood size is 1x1 then operation is called intensity transformation or point processing as output of function is dependent only on that pixel.

here are some example of intensity transformation

  1. image negative
  2. log transformation
  3. power law transformation /gamma transformation
  4. piece wise linear transformation

1. Image Negative :

Operation for this is : T (r) = (L - 1) - r

Let suppose we have an RGB color image which has 24 bits per pixel value . 8 bit per pixel for each channel then we will subtract all channel intensity from 255 to get negative .

Smoothing :

Smoothing is used to make images less pixelated or to lessen noise. Although low-pass filters are the foundation of the majority of smoothing techniques, we can also smooth an image by using the average or median value of a group of pixels (a kernel) that moves across the image.

Blur :

If a blurred image is carefully examined, a we found out that the image is smooth, which means that edges are not visible. Low pass filters are used to blur images because they let low frequency waves in while blocking out high frequency ones. Frequency in this context refers to pixel value changes. Since the blurred image is smooth around the edges, high frequency should be filtered out.

Because a pixel value needs to be close to its neighbour value in order to blur an image, a filter with every call having value 1 is used for this purpose.

Since the goal is to reduce contrast, the filter divides the value of each pixel by 9 to maintain normalisation.

Gaussian Blur :

This filter gives different weight to each entries in matrix as entries in kernel. The closer pixel to the selected pixel has greater weight while the further pixel has lower weight.

Moreover, this kernel cannot be sufficiently large because the further pixel give smaller contribution to the value of kernel. Sharpening :

In image processing, sharpening refers to a technique used to enhance the details and edges in an image, making it appear more visually clear. It is usually used to improve the overall quality and impact of images.

The process of sharpening involves emphasizing the high-frequency components in an image, which correspond to the edges and fine details. The basic idea is to enhance the contrast between adjacent pixels along the edges, making them more different.

Here’s a step-by-step explanation of the sharpening process in image processing.

We have to convert the image to grayscale if it’s not already in that format. This step simplifies the sharpening process as it focuses on the brightness values of the image.

A sharpening kernel is a filter that is applied to the image. It defines how the pixel values in the image will be modified to achieve the sharpening effect. The kernel typically has negative values at its centre, surrounded by positive values. The negative centre value enhances the edges and details, while the positive values blend the surrounding pixels to create a smooth transition.

Apply the sharpening kernel to the grayscale image. This process involves convolving the kernel with each pixel in the image to calculate the new pixel values. The convolution operation calculates the weighted sum of the kernel values and the corresponding pixel values.

The resulting sharpened image may have amplified noise and artifacts due to the high-frequency emphasis. To less these effects, you can adjust the pixel values. Techniques such as clipping the pixel values to a specific range, applying a threshold, or using noise reduction algorithms can be employed to improve the image quality.

The process of sharpening involves increasing the contrast around edges, which makes them ap- pear more pronounced.

Here,we first convert image in frequency domain then we perform operation on it and convert result into spatial domain .
