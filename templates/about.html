About Our Project

This Project is based on Digital Image Processing & Image Encryption using chaotic functions and DNA Encoding. Since Digital Image Processing is a vast field, we explored many topics and techniques to modify given images. From basics of Image to Image Encryption Algorithm, we explored Many things. We created learn section such that anyone can learn this techniques and to apply them we created tools section. All thanks to Pro. Manish Gupta, Pro. Prosenjit Kundu & Pro. Manoj Raut
